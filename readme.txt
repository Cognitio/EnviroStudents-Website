Hi!

This folder contains all the code and resources for the ECHS Website.
---------------------------------------------------------------------

Open index.html for the homepage.

The other html files are for the other webpages including the work folder which holds project pages. (can be easily appended to)

The css folder contains a unique styling for the homepage, a shared styling for the other webpages, and a resource
that enables a modern look for the pages.

The top level js folder contains a script for parallax scrolling.

The images folder contains all pictures used.

The calendar and weatherAssets folders contain the code for the event calendar and weather widget with their 
own respective styling and javascript.
